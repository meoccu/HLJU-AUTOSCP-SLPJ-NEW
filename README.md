# 黑龙江大学自动评教助手

> 油猴/脚本猫自动化脚本 · 适用于黑龙江大学教育质量监控系统 `zlpj.hlju.edu.cn`

自动进入问卷、判断题选"是"、量表题选满分、主观题填评语、自动提交、循环遍历全部待评课程。支持手动核验/全自动双模式。

---

三种方式任选其一：

| 方式 | 链接 | 特点 |
|------|------|------|
| 🐙 GitHub | [meoccu/HLJU-AUTOSCP-SLPJ-NEW](https://github.com/meoccu/HLJU-AUTOSCP-SLPJ-NEW) | 一键安装，自动更新 |

| 📋 手动源码 | 见下方 `HLJU-PingJiao-Auto.user.js` | 复制源码粘贴到油猴，离线可用 |

### 手动安装步骤

1. 浏览器安装 [Tampermonkey](https://www.tampermonkey.net/)（油猴）插件
2. 点击油猴图标 → 管理面板 → 新建脚本
3. 删除默认内容，粘贴 `HLJU-PingJiao-Auto.user.js` 全部源码
4. `Ctrl + S` 保存，确认开关为绿色（启用）
5. 打开 `zlpj.hlju.edu.cn` 登录，进入评教列表页
6. 右上角出现配置面板，点击「开始评教」或按 `Ctrl+Shift+A` 启动

## 功能特性

- **自动巡检**：扫描 `.survery .item.bottom` 列表项，识别未评教课程（无 `.icon-defen` 标记）
- **自动进入**：点击课程卡片进入问卷页（基于真实 hash 路由 `#/survery/children/surverySubmit/`）
- **自动作答**：判断题选"是"，量表题选满分（5分/4分可切换），主观题填两段不同评语
- **自动提交**：填写完成后自动点击提交按钮，处理 weui-dialog 二次确认弹窗
- **循环遍历**：提交成功后返回列表，自动处理下一门，直到全部完成
- **断点续评**：localStorage 记录已完成问卷，刷新不重复
- **双模式**：手动模式（填完暂停等核验）/ 全自动模式
- **防风控**：随机延时模拟人工操作
- **面板智能显隐**：只在评教页面显示，进入课表等其他页面自动隐藏

## 自动作答规则

| 题型 | 题号 | 自动选项 |
|------|------|----------|
| 判断题 | 1-2 | 固定选"是" |
| 量表题 | 3-9 | 选满分5分（可切换良好4分） |
| 主观题 | 10 | 填入"最满意方面"评语 |
| 主观题 | 11 | 填入"需要改进方面"评语 |

默认评语：

- **最满意**：老师讲课条理清晰，知识点讲解通俗易懂，课堂节奏把控到位，课后答疑耐心细致，教学态度认真负责，收获非常大。
- **需改进**：建议可以多补充一些典型例题，拓宽习题讲解范围，进一步丰富线上教学资源，增加课堂互动环节。

评语可在面板中自定义编辑，localStorage 持久化保存。

## 常见问题

<details>
<summary><b>面板没出现？</b></summary>

- 确认油猴脚本已启用（绿色开关）
- 确认当前在评教页面（hash 含 `/survery`），课表等其他页面不显示面板
- 按 F12 控制台查看是否有 `[自动评教]` 日志
</details>

<details>
<summary><b>脚本不填写题目？</b></summary>

等待问卷页 DOM 渲染完成（约800ms），脚本会自动尝试10次。若仍失败，可能是页面结构改版，需检查 DOM 选择器。
</details>

<details>
<summary><b>提交失败？</b></summary>

脚本会自动处理 weui-dialog 确认弹窗。若弹出验证码，需手动完成，脚本会暂停等待。
</details>

<details>
<summary><b>如何重新评教？</b></summary>

清除浏览器 localStorage 中的 `hlju_pj_done_v2` 记录，或重新安装脚本。
</details>

1. 所有评教评分使用正向客观评语，不存在恶意低分

3. 模拟人工操作节奏，不对服务器产生压力
4. 请在评教开放期内合法使用
